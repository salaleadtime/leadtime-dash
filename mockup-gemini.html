Insira sua chave do Google AI Studio para análise com Gemini.

Como obter a chave gratuita do Gemini

  1. Acesse aistudio.google.com e faça login com sua conta Google.
  2. Clique em Get API Key → Create API Key.
  3. Copie a chave gerada (começa com AIzaSy...).
  4. Cole no campo acima e clique em Salvar e analisar.
  5. A chave fica salva no seu browser — você não precisará informar novamente.

Segurança da chave

  • ✓A chave é salva apenas no localStorage do browser local — não sobe para nenhum servidor.
  • ✓Cada usuário usa sua própria chave gratuita do Google.
  • !Não compartilhe sua chave. Cada conta Google tem cota própria (1.500 req/dia grátis).

Diferença entre regras e IA

  • ✕Regras: "3 pontos de atenção e 2 itens sem responsável" — texto genérico, igual para qualquer sprint com esses números.

Como o cache funciona

  • ✓A chave do cache é gerada com base em: squad + quantidade de histórias + nº de bloqueios + nº de dependências + nº em homologação + nº em deploy.
  • ✓Se qualquer um desses números mudar (nova importação), o cache é invalidado e a próxima análise vai ao Gemini.
  • ✓Válido por 30 minutos. Após isso, a próxima chamada busca análise nova automaticamente.
  • →O botão "Atualizar análise" força uma nova chamada ao Gemini mesmo com cache válido.
